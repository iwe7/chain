## `class decorator` [装饰类]

```ts
(target:any)=> return class extends target{}
```

```ts

```

## `constructor decorator` [装饰构造函数]

```ts
() => any;
```

## `parameter decorator` [装饰参数]

```ts
() => any;
```

## `method decorator` [方法]

